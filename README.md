# Tab Group Bookmarks

A Chrome extension that saves tab groups as bookmark folders.

## Features

- Lists all tab groups in the current window
- Saves all tabs in a group as bookmarks inside a chosen folder
- Skips duplicate URLs automatically
- Prompts for a name if the group is unnamed

## Usage

1. Click the extension icon in the toolbar
2. Select a tab group to save
3. Choose a destination bookmark folder
4. Click **Save Here**

## Permissions

| Permission | Reason |
|------------|--------|
| `tabGroups` | Read tab groups in the current window |
| `tabs` | Read tab URLs and titles |
| `bookmarks` | Create bookmark folders and bookmarks |
